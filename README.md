# Airline Data Analysis Dashboard
This project is an interactive Power BI dashboard developed to analyze airline operational performance and flight statistics. 
The dashboard provides insights into flight operations, delay patterns, and cancellation trends to support data-driven decision-making.
# Tools Used
Microsoft Power BI
Microsoft Excel / CSV
Power Query
DAX
# Key Matrices
Total Flights
Total Cancelled
Arrival Delay
Departure Delay
Cancellation Rate
On-Time Performance
